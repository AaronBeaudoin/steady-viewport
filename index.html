⛰ steady-viewport Demo

Showcasing a minimal flexible layout helped by steady-viewport. This "top height" hero section is sized using the --th CSS variable for pixel-perfect height. Its height is simply --vh minus the height of any element at the top of the document/page marked with the data-steady="top" or data-steady="sticky" attributes.

height: var(--ph);

This "page height" section is sized using the --ph CSS variable for pixel-perfect height. Its height is simply --vh minus the height of any elements in the document marked with the data-steady="sticky" attribute.

If the height appears to be too short on mobile, remember that the mobile URL navigation bar or other UI elements shrinking has no effect on --vh.

height: var(--bh);

This "bottom height" section is sized using the --bh CSS variable for pixel-perfect height. Its height is simply --vh minus the height of any element at the bottom of the document/page marked with the data-steady="bottom" or data-steady="sticky" attributes.
